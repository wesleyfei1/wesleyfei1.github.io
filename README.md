# Weihan Fei's Academic Homepage

This is the source code for my academic personal website: [https://wesleyfei1.github.io](https://wesleyfei1.github.io)

Built with [AcademicPages](https://academicpages.github.io/), a Jekyll theme for academic personal websites.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`.
